

功能一览：


\begin{DoxyItemize}
\item 1.\+3\+MP 摄像头，能够以以下分辨率拍摄视频
\begin{DoxyItemize}
\item S\+X\+GA (1280 x 1024)：最高 15 fps（帧/秒）
\item V\+GA (640 x 480)：最高 30 fps
\item C\+IF (352 x 288)：最高 60 fps
\item Q\+V\+GA (320 x 240)：最高 60 fps
\item Q\+C\+IF (176 x 144)：最高 120 fps
\item Q\+Q\+V\+GA (160 x 120)：最高 60 fps
\item Q\+Q\+C\+IF (88 x 72)：最高 120 fps
\item 滚动快门，\+F2.8，65 度水平视野。
\end{DoxyItemize}
\item 四核 A\+RM Cortex A7 处理器，默认时钟频率 1.\+35\+G\+Hz。支持硬浮点运算 (V\+F\+Pv4) 和 N\+E\+ON S\+I\+MD 多媒体指令。
\item 双核 M\+A\+L\+I-\/400 G\+P\+U（图形处理单元），支持 Open\+G\+L-\/\+ES 2.\+0。
\item 256MB D\+D\+R3-\/1600 S\+D\+R\+A\+M。
\item U\+SB 供电（需要 U\+SB 3.\+0 端口或 Y 型电缆连接到两个 U\+SB 2.\+0 端口）。最大功耗为 3.\+5 瓦，通常为 1.\+5 瓦 -\/ 2.\+5 瓦，具体取决于 Je\+Vois 处理器上运行的机器视觉算法。
\item 集成冷却风扇确保 C\+PU 和 G\+PU 即使在持续满处理负荷下也能以最大频率运行（系统芯片通常在满负荷下连续几天或几周以低于 50C 的温度运行，而开始降低时钟速度以避免处理器过热的第一个阈值是 75C）。
\item Mini-\/\+U\+SB 2.\+0 B 型 5 针端口，用于电源和视频数据。
\item U\+SB 视频类 (U\+VC) 兼容接口到主机，\+Je\+Vois 平台显示为标准 U\+SB 网络摄像头，无需在 Windows、\+Mac\+O\+S、\+Android 和 Linux 上安装驱动程序（i\+OS 检测到该设备，但从 i\+OS 9 开始似乎不支持网络摄像头类设备，无论是来自 Je\+Vois 还是我们测试的其他制造商）。
\item 双电压串行端口（\+U\+A\+RT 5V 或 3.\+3V，通过 V\+C\+C-\/\+IO 引脚选择），允许设备直接与 Arduino 或其他嵌入式控制器通信。
\item Micro-\/\+SD 插槽可容纳嵌入式操作系统、\+Je\+Vois 机器视觉模块，并可存储用户数据（例如设备捕获的视频）。
\item 一个双色 L\+E\+D：绿色：电源良好（\+U\+SB 端口的功率还不足以为设备供电）。橙色：电源良好，摄像头正在传输视频帧。
\item 塑料外壳带有 4 个孔，用于安全安装。
\end{DoxyItemize}
