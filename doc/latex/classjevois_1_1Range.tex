通用 \mbox{\hyperlink{classjevois_1_1Range}{Range}} 类 

\mbox{\hyperlink{classjevois_1_1Range}{Range}} 类用于表示从 \mbox{[}min .. max\mbox{]} 的范围

请注意，\+Range 的运算符不使用类型提升。这是为了最大限度地减少模板负担，也是为了减少程序员的困惑。例如，虽然原则上可以定义 Range$<$int$>$ + float 并返回 Range$<$float$>$，但这里我们没有定义这样的运算符。通过首先转换范围然后添加常量可以实现相同的结 果：


\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classjevois_1_1Range}{jevois::Range<int>}} r1(0, 100);}
\DoxyCodeLine{\textcolor{comment}{// NOT ALLOWED: jevois::Range<float> r2 = r1 + 1.23F;}}
\DoxyCodeLine{\mbox{\hyperlink{classjevois_1_1Range}{jevois::Range<float>}} r2 = \mbox{\hyperlink{classjevois_1_1Range}{jevois::Range<float>}}(r1) + 1.23F; \textcolor{comment}{// OK, 没有隐藏转换，易于阅读，明确}}
\end{DoxyCode}


\mbox{\hyperlink{classjevois_1_1Range}{Range}} 上的运算符在内部使用范围检查和限制。因此，使用 Range$<$byte$>$ 时要小心。例如： 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classjevois_1_1Range}{jevois::Range<jevois::byte>}} r(jevois::byte(0), jevois::byte(250));}
\DoxyCodeLine{r += jevois::byte(100); \textcolor{comment}{// new range is 100-\/255}}
\end{DoxyCode}
