基于正则表达式的有效值规范，与正则表达式匹配的所有内容均被视为有效 

在内部使用 boost\+::regex（因为 std\+::regex 不允许从正则表达式中获取原始字符串规范，但我们需要它来显示帮助消息）。 这允许高度灵活的有效值定义。例如，假设您希望 int 参数在范围 \mbox{[}0..59\mbox{]} 内，但它也可以具有值 72，则您的正则表达式 将是：

\begin{DoxyVerb}^(([0-5]?[0-9])|72)$
\end{DoxyVerb}


您可以在网络上找到正则表达式示例来匹配有效文件名、有效 U\+R\+L、有效信用卡号等。只需确保您的正则表达式符合 boost\+::regex 所期望的语法，因为正则表达式有几种语法。
