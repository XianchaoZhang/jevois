包装 Python 使用的 \mbox{\hyperlink{classjevois_1_1InputFrame}{Input\+Frame}} 

此包装器是为了解决我们的 Python 支持中缺少移动语义的问题。此类不适用于一般用途，仅供 \mbox{\hyperlink{classjevois_1_1PythonModule}{Python\+Module}} 使用。此类的用户必须确保原 始 \mbox{\hyperlink{classjevois_1_1InputFrame}{Input\+Frame}} 将比任何和所有 \mbox{\hyperlink{classjevois_1_1InputFramePython}{Input\+Frame\+Python}} 实例都存活得更久，因为 \mbox{\hyperlink{classjevois_1_1InputFramePython}{Input\+Frame\+Python}} 只是通过不受保护的原始指针引用源 Input\+Frame。虽然 C++ 对象称为 Input\+Frame\+Python，但我们将以 \mbox{\hyperlink{classjevois_1_1InputFrame}{Input\+Frame}} 的名称将其公开给 python（请参阅 Python\+Support.\+C）。
