Python 使用的 \mbox{\hyperlink{classjevois_1_1OutputFrame}{Output\+Frame}} 包装器 

此包装器是为了解决我们的 Python 支持中缺少移动语义的问题。此类不适用于一般用途，而仅供 \mbox{\hyperlink{classjevois_1_1PythonModule}{Python\+Module}} 使用。此类的用户必须确保 原始 \mbox{\hyperlink{classjevois_1_1OutputFrame}{Output\+Frame}} 将比任何和所有 \mbox{\hyperlink{classjevois_1_1OutputFramePython}{Output\+Frame\+Python}} 实例都存活得更久，因为 \mbox{\hyperlink{classjevois_1_1OutputFramePython}{Output\+Frame\+Python}} 只是通过不受保护的原始指针引用 源 Output\+Frame。虽然 C++ 对象称为 Output\+Frame\+Python，但我们将以名称 \mbox{\hyperlink{classjevois_1_1OutputFrame}{Output\+Frame}} 将其公开给 python（参见 Python\+Support.\+C）。
