Backend for Video\+Display on Je\+Vois-\/\+Pro. 

This abstract base class provides an interface used by \mbox{\hyperlink{classjevois_1_1VideoDisplayGL}{Video\+Display\+GL}}. Derived implementations are available for X11+\+Open\+G\+L-\/\+ES 3.\+2 (to be used on Je\+Vois-\/\+Pro host) and Mali framebuffer + Open\+G\+L-\/\+ES 3.\+2 (to be used on Je\+Vois-\/\+Pro platform).
