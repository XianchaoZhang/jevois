视频输入的基类，将派生为 \mbox{\hyperlink{classjevois_1_1Camera}{Camera}} 和 \mbox{\hyperlink{classjevois_1_1MovieInput}{Movie\+Input}} 

\mbox{\hyperlink{classjevois_1_1Engine}{Engine}} 使用 \mbox{\hyperlink{classjevois_1_1VideoInput}{Video\+Input}} 捕获输入帧并将它们传递给其当前加载的机器视觉模块进行处理。\+Video\+Input 类是抽象的，仅定义接 口。对于实时视频处理，\+Engine 将其 \mbox{\hyperlink{classjevois_1_1VideoInput}{Video\+Input}} 实现为 Camera，对于预先录制的视频或图像序列的处理（可用于调试或调整 可重现图像序列的算法），它还可以将其 Video\+Inout 实现为 Movie\+Input。
