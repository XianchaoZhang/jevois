\hypertarget{UserConnect_autotoc_md118}{}\doxysubsection{基础知识}\label{UserConnect_autotoc_md118}
Je\+Vois-\/\+A33智能相机只有两个连接器：


\begin{DoxyItemize}
\item 一个 mini-\/\+U\+SB 2.\+0 B 型 5 针连接器，用于供电（必需）和将视频传输到主机（可选，取决于 Je\+Vois 内部运行哪个机器视觉模块）；
\item 一个 4 针 J\+S\+T-\/\+SH 1.\+0 毫米连接器（称为{\itshape 微串行}连接器），用于与微控制器（例如 Arduino）进行可选 T\+TL 级串行 (U\+A\+RT) 连接。
\end{DoxyItemize}

始终需要通过 mini-\/\+U\+SB 连接器供电才能运行。您应努力提供以下信息：


\begin{DoxyItemize}
\item 调节5V +/-\/ 10电压。
\item 电流不低于 800m\+A（因此，总功率为 4 瓦）。如果您知道您将只使用一些不会完全加载 Je\+Vois 内部 C\+PU 的机器视觉算法，那么电流可能会更小，因此会产生较低的功率要求。
\item 没有显著的电流和功率通过微型串行端口（有关更多详细信息，请参阅 User\+Serial）。因此，您无法通过微型串行端口为 Je\+Vois 供电，而始终需要通过 Mini-\/\+U\+SB 端口为其供电。
\end{DoxyItemize}

可能存在多种电源和数据场景。下面给出了几个示例。\hypertarget{UserConnect_autotoc_md119}{}\doxysubsection{与主机的标准连接}\label{UserConnect_autotoc_md119}

U\+SB 2.\+0 端口的最大额定电流输出为 500m\+A（5V 时）。请注意，随着电流使用量的增加，电压可以降至 5V 以下。因此，将 Je\+Vois 连接到单个 U\+SB 2.\+0 端口可能无法提供足够的电力来在各种机器视觉算法下操作 Je\+Vois。以前曾用于为需要超过 500mA 电流的外部硬盘或 C\+D-\/\+Rom 供电的标准技巧是使用所谓的 {\bfseries{Y 型}} 电缆，该电缆连接到两个 U\+SB 端口，从而将两个端口的电源组合起来，从而为设备提供高达 1A 的电流。Y 型电缆如下所示：

\hypertarget{UserConnect_autotoc_md120}{}\doxysubsection{U\+S\+B 电缆类型}\label{UserConnect_autotoc_md120}
有许多类型的 mini-\/\+U\+SB 2.\+0 电缆。最便宜的电缆通常使用能够传输数据的最细的电线。这些电缆可能不适合为 Je\+Vois 传输足够的电力，因为非常细的电线无法为 Je\+Vois 传输足够的电流。

在美国，铜线直径使用 \href{https://en.wikipedia.org/wiki/American_wire_gauge}{\texttt{ 美国线规}} 或简称 A\+WG 来测量。\+A\+WG 值越小，表示电线越粗，可以承载更多电流，损耗越小。

最便宜的 U\+SB 电缆可能使用 30A\+WG 或 32A\+WG 线径（非常细）。虽然这对于数据线来说很合适，但不足以为 Je\+Vois 供电。

如果您购买 mini-\/\+U\+SB 线缆，请确保线缆内部的两根电源线的线径为 24A\+WG 或更低（这些线缆的两根数据线的线径通常为 28A\+W\+G，因此会标记为 24/28\+A\+WG 线缆）。有时线缆上会写有以下信息：
