包装 \mbox{\hyperlink{classjevois_1_1GUIhelper}{G\+U\+Ihelper}} 以供 Python 使用 

此类不适用于一般用途，仅供 \mbox{\hyperlink{classjevois_1_1PythonModule}{Python\+Module}} 使用。此类的用户必须确保原始 \mbox{\hyperlink{classjevois_1_1GUIhelper}{G\+U\+Ihelper}} 将比任何和所有 \mbox{\hyperlink{classjevois_1_1GUIhelperPython}{G\+U\+Ihelper\+Python}} 实例存活的 时间更长，因为 \mbox{\hyperlink{classjevois_1_1GUIhelperPython}{G\+U\+Ihelper\+Python}} 只是通过不受保护的原始指针引用源 G\+U\+Ihelper。虽然 C++ 对象称为 G\+U\+Ihelper\+Python，但我们将以 \mbox{\hyperlink{classjevois_1_1GUIhelper}{G\+U\+Ihelper}} 的名称将其公开给 python（参见 Python\+Support.\+C）。
