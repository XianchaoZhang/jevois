Simple class to monitor a serial port in the Je\+Vois-\/\+Pro G\+UI. 

This class opens a window and displays traffic on a given serial port. Its interface is the same as a regular \mbox{\hyperlink{classjevois_1_1Serial}{Serial}} (or its parent, \mbox{\hyperlink{classjevois_1_1UserInterface}{User\+Interface}}), with the addition of a \mbox{\hyperlink{classjevois_1_1GUIserial_a6fdfd59976c1605f3c5ad0dda4a19629}{draw()}} function to render into a window managed by the caller.
