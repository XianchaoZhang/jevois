除了用 C++ 编写模块外，\+Je\+Vois 还支持用 Python 编写模块。\+Je\+Vois 提供双向映射： 


\begin{DoxyItemize}
\item 通过导入 Python 包 {\ttfamily libjevois} ，可以在 Python 中访问 Je\+Vois 的 C++ 函数和类
\item Je\+Vois 引擎可以直接调用实现机器视觉处理模块的 Python 类的类成员函数
