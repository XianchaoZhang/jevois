\begin{DoxyItemize}
\item 从查看 \mbox{\hyperlink{classjevois_1_1Component}{Component}} 开始，它强调了如何在 Je\+Vois 中组装复杂视觉处理管道的广泛概念框架，并大量重复使用共享算法元素。

\item 最后看一下 Manager，它是一个特殊组件，是组件层次结构的根。
\item 然后，您将通过查看文档模块中的其他元素来准备进入下一个级别。
