一个通用的 singleton 类，用于强制一个对象的单个实例 

\mbox{\hyperlink{classjevois_1_1Singleton}{Singleton}} 类应该被任何需要的类继承：
\begin{DoxyEnumerate}
\item 只有一个实例
\item 可全局访问
\end{DoxyEnumerate}

请注意，目前无法保证 \mbox{\hyperlink{classjevois_1_1Singleton}{Singleton}} 的析构顺序，因此应注意不要在 \mbox{\hyperlink{classjevois_1_1Singleton}{Singleton}} 的析构函数中引用另一个 Singleton。

这个特定的实现来自：http\+://stackoverflow.com/questions/1008019/c-\/singleton-\/design-\/pattern
